## 1. 项目概述

### 1.1 项目目标
创建一个完整的种子源模块从机设备模拟器，用于测试和调试现有的种子源模块上位机软件。该模拟器将：
- 实现完整的通信协议
- 提供友好的GUI界面
- 模拟真实设备的行为
- 记录通信过程便于调试

### 1.2 参考协议
基于《种子源模块通信协议V1.3》规范实现

### 2.2 设计原则
- **独立项目**：作为独立的Qt应用程序开发

## 3. 功能模块设计

### 3.1 串口通信层

**职责**：管理串口连接，监听串口数据

**功能**：
- 串口配置（端口、波特率、数据位、停止位、校验位、流控制）
- 数据接收与发送
- 连接状态管理与通知
- 错误处理与自动重连选项

**主要类**：`SimulatorWorker`

### 3.2 协议处理器

**职责**：解析接收到的协议帧，执行对应命令，构建响应帧

**功能**：
- 帧解析（帧头/帧尾/校验和验证）
- 命令分发（读寄存器/写寄存器/读状态/设备控制）
- 响应帧构建
- 错误响应处理

**主要类**：`SimulatorProtocol`

### 3.3 寄存器管理器

**职责**：管理所有寄存器状态，模拟真实设备行为

**功能**：
- 存储所有寄存器的当前值
- 寄存器读写接口
- 设备状态模拟（启动/停止/运行中）
- 简单的物理模拟（电流/温度/功率随时间变化）

**主要类**：`SimulatorRegisterManager`

### 3.4 GUI界面

**职责**：提供用户交互界面

**功能**：
- 串口配置面板
- 通信日志显示（十六进制+时间戳）
- 寄存器状态显示与编辑
- 设备状态显示

**主要类**：
- `MainWindow` - 主窗口
- `SerialConfigPanel` - 串口配置面板
- `LogPanel` - 通信日志面板
- `RegisterPanel` - 寄存器状态面板
- `StatusPanel` - 设备状态面板

## 4. 数据流程设计

### 4.1 命令处理流程

```
上位机发送命令
      │
      ▼
串口接收数据
      │
      ▼
缓冲区收集完整协议帧
      │
      ▼
帧验证（帧头/帧尾/校验和）
      │
      ├─→ 验证失败 → 丢弃/记录错误
      │
      ▼
验证成功
      │
      ▼
解析命令类型
      │
      ├─→ 读寄存器命令 → 读取寄存器值 → 构建响应帧
      ├─→ 写寄存器命令 → 更新寄存器值 → 构建响应帧
      ├─→ 读状态命令 → 获取当前状态 → 构建响应帧
      └─→ 设备控制命令 → 更新设备状态 → 构建响应帧
      │
      ▼
发送响应帧
      │
      ▼
记录日志+GUI更新
```

### 4.2 协议帧格式

**请求/响应帧格式：**
```
[帧头 0xAA][地址码][命令码][数据长度 L][数据*L][校验和][帧尾 0x55]
```

**校验和计算：** 从帧头到数据部分的累加和，取低8位

## 5. 错误处理策略

| 错误类型 | 处理方式 |
|---------|---------|
| 帧验证失败（帧头/帧尾错误） | 记录错误日志，不发送响应 |
| 校验和错误 | 记录错误日志，不发送响应 |
| 未知命令 | 返回错误响应帧 |
| 寄存器不存在 | 返回错误响应帧 |
| 写入值超出范围 | 返回错误响应帧 |
| 串口错误 | 显示错误提示，提供自动重连选项 |

## 6. 设备状态模拟

### 6.1 状态机

```
待机
  │
  ├─→ 启动命令 → 启动中
  │                 │
  │                 └─→ 运行中
  │
运行中
  │
  ├─→ 停止命令 → 停止中
  │                 │
  │                 └─→ 待机
  │
  └─→ 报警条件触发 → 报警状态
```

### 6.2 数值模拟

- **电流**：在设置目标值附近有±2%的随机波动
- **温度**：随电流大小缓慢变化，有热惯性
- **功率**：基于电流计算，P=I²R
- 启动后数值会平滑过渡到目标值

## 8. 寄存器定义

| 基地址 | 偏移 | 名称 | 大小 | 类型 | 说明 |
|-------|------|------|------|------|------|
| 0x00 | 0x00 | STATUS | 4 | uint32 | 设备状态寄存器 |
| 0x01 | 0x00 | ALERT | 4 | uint32 | 报警状态寄存器 |
| 0x02 | 0x00 | CURRENT | 4 | uint32 | 电流控制寄存器 (mA) |
| 0x03 | 0x00 | TEMP | 4 | int32 | 温度控制寄存器 (°C) |
| 0x04 | 0x00 | POWER | 4 | uint32 | 功率控制寄存器 (mW) |
| 0x05 | 0x00 | CONFIG | 4 | uint32 | 配置寄存器 |
| 0x06 | 0x00 | SYSTEM | 4 | uint32 | 系统控制寄存器 |
| 0x07 | 0x00 | DEVINFO | 4 | uint32 | 设备信息寄存器 |
